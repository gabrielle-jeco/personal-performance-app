How to run:
- clone repo dengan command 'git clone https://github.com/gabrielle-jeco/personal-performance-app.git'
- build image docker dan jalankan dengan command 'docker-compose up -d --build' (pastikan Docker desktop jalan/running)
- akses website di localhost:8080
